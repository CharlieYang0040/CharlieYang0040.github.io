나만의 툴 만들기 (기초 UI)

cmds.window(), cmds.button() 등의 명령어로 사용자가 클릭할 수 있는 간단한 UI 창을 만들 수 있습니다. 버튼의 command에 함수를 연결하는 것이 핵심입니다.

[기본 예제]

import maya.cmds as cmds

def create_cube(*args):
    cmds.polyCube()

# UI가 이미 있다면 삭제하고 새로 만듭니다.
if cmds.window("MyFirstUITool", exists=True):
    cmds.deleteUI("MyFirstUITool")

cmds.window("MyFirstUITool", title="나의 첫 UI 툴")
cmds.columnLayout()
cmds.button(label="큐브 생성하기", command=create_cube)
cmds.showWindow("MyFirstUITool")

[실전 문제]

위 예제 코드에 '스피어 생성하기' 버튼을 추가해 보세요. 이 버튼을 누르면 `cmds.polySphere()`가 실행되어야 합니다.

해답:

import maya.cmds as cmds

def create_cube(*args): cmds.polyCube()
def create_sphere(*args): cmds.polySphere()

if cmds.window("MyObjectCreator", exists=True):
    cmds.deleteUI("MyObjectCreator")

cmds.window("MyObjectCreator", title="오브젝트 생성기")
cmds.columnLayout()
cmds.button(label="큐브 생성하기", command=create_cube)
cmds.button(label="스피어 생성하기", command=create_sphere)
cmds.showWindow("MyObjectCreator")
