함수를 활용한 스크립트 정리

특정 기능을 수행하는 코드 덩어리를 함수(Function)로 만들어두면, 필요할 때마다 재사용할 수 있어 스크립트가 깔끔해지고 관리하기 쉬워집니다.

[기본 예제]

import maya.cmds as cmds

def create_lamppost():
    """가로등을 생성하는 함수"""
    pillar_info = cmds.polyCylinder(name="Pillar", ch=True)
    light_name = cmds.polyCube(name="Light")[0]
    cmds.setAttr(f"{pillar_info[1]}.height", 5)
    cmds.setAttr(f"{light_name}.translateY", 2.5)

# 함수 호출만으로 가로등 생성
create_lamppost()

[실전 문제]

`create_lamppost` 함수를 수정하여, 가로등의 높이(height)와 전등의 크기(light_size)를 함수를 호출할 때 직접 지정할 수 있도록 만들어보세요.

해답:

import maya.cmds as cmds

def create_lamppost(height, light_size):
    pillar_node = cmds.polyCylinder(ch=True)[1]
    light_name = cmds.polyCube(w=light_size, h=light_size, d=light_size)[0]
    
    cmds.setAttr(f"{pillar_node}.height", height)
    cmds.setAttr(f"{light_name}.translateY", height / 2.0)

# 높이 7, 전등 크기 1.5인 가로등 생성
create_lamppost(height=7, light_size=1.5)
