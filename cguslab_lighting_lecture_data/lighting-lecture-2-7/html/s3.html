여러 오브젝트 한번에 제어하기

cmds.ls(sl=True)로 현재 선택된 오브젝트 리스트를 얻고, `for` 반복문으로 각 오브젝트에 동일한 작업을 수행하여 강력한 자동화 스크립트를 만들 수 있습니다.

[실전 문제 (라이팅 아티스트용)]

선택한 여러 개의 가로등 에셋마다 중앙 상단에 포인트 라이트를 자동으로 추가하고, 해당 가로등에 종속(parent)시키는 코드를 작성해보세요.

힌트: cmds.xform(query=True, boundingBox=True)를 사용하면 오브젝트의 경계 상자(크기와 위치) 정보를 얻을 수 있습니다.

해답:

import maya.cmds as cmds

lampposts = cmds.ls(sl=True, type='transform')

if not lampposts:
    cmds.warning("먼저 가로등 에셋들을 선택해주세요.")
else:
    for post in lampposts:
        # 바운딩 박스로 오브젝트의 월드 공간상 크기와 위치를 얻음
        bbox = cmds.xform(post, q=True, bb=True, ws=True)
        # bbox 결과: [xmin, ymin, zmin, xmax, ymax, zmax]
        
        # X, Z의 중앙, Y의 최대값을 라이트 위치로 지정
        light_pos = [ (bbox[0]+bbox[3])/2, bbox[4], (bbox[2]+bbox[5])/2 ]
        
        new_light = cmds.pointLight(name=f"{post}_pointLight")
        cmds.move(light_pos[0], light_pos[1], light_pos[2], new_light)
        
        # 생성된 라이트를 가로등 그룹에 부모-자식 관계로 묶음
        cmds.parent(new_light, post)
        
        print(f"'{post}'에 라이트를 추가하고 종속시켰습니다.")
