더 효율적인 코딩 (PyMEL)

PyMEL(pymel.core)은 더 현대적인 파이썬 스타일(객체 지향)로 Maya를 다룰 수 있게 해주는 라이브러리입니다. 코드가 더 짧고 직관적으로 변하는 장점이 있습니다.

[기본 예제]

# maya.cmds 방식
import maya.cmds as cmds
cmds.setAttr("pCube1.translateX", 5)

# PyMEL 방식
import pymel.core as pm
my_cube = pm.PyNode('pCube1')
my_cube.translateX.set(5) # 오브젝트의 속성에 직접 접근

[실전 문제]

2단계의 실전 문제였던 '탑 쌓기'를 이번에는 PyMEL을 사용하여 작성해 보세요.

해답:

import pymel.core as pm

for i in range(5):
    # 큐브를 생성하면 바로 PyMEL 노드(객체)로 반환됩니다.
    new_cube = pm.polyCube()[0]

    # 오브젝트의 속성에 직접 접근하여 위치를 설정합니다.
    new_cube.translateY.set(i)
