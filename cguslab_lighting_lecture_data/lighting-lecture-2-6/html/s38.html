예제 코드로 매개변수 익히기 2: 원하는 색상으로 라이트 변경하기

1주차에 배운 라이트 색상 변경 코드를 재사용 가능한 함수로 업그레이드합니다.

어떤 라이트(light_name)의 색을 어떤 RGB 값(r, g, b)으로 바꿀지 함수 호출 시 자유롭게 지정할 수 있습니다.

[실행 코드]

import maya.cmds as mc

def set_light_color(light_name, r, g, b):
    # light_name에 해당하는 라이트의 color 속성을 (r, g, b) 값으로 변경
    # light_name은 'pointLightShape1'과 같은 쉐입 노드 이름이어야 합니다.
    mc.setAttr(light_name + ".color", r, g, b, type='double3')
    print(f"{light_name}의 색상을 ({r}, {g}, {b})로 변경했습니다.")

# --- 사용법 ---
# 씬에 pointLight와 spotLight를 만들고 아래 함수를 각각 호출해보세요.
# 쉐입 노드의 이름을 정확히 적어주어야 합니다.
