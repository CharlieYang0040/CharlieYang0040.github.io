정규표현식 심화: 캡처 그룹과 전후방탐색

복잡한 텍스트에서 정확한 정보만 추출합니다. 캡처 그룹((), (?P<name>...)), 비캡처 그룹((?:...)), 전/후방탐색((?<=...), (?=...))을 활용합니다.

[기본 예제]

import re

text = "asset_v012.ma\nasset_v013_wip.ma\nemail: user@test.com\n\nHello\n"

# 버전 태그 추출
pat = re.compile(r"\b(?P[\w-]+)_v(?P\d{3})(?P_wip)?\.(?:ma|mb)\b", re.I)
print([m.groupdict() for m in pat.finditer(text)])

# 태그 내용 비탐욕 추출
m = re.search(r"(.*?)", text, flags=re.S)
print(m.group(1).strip() if m else None)

# 이메일 도메인 마스킹
print(re.sub(r"(?<=@)[\w.-]+", "******", text).splitlines()[2])

[실습]

로그에서 라인 형태 [LEVEL] YYYY-MM-DD hh:mm:ss tag=XYZ user=NAME msg=...를 모두 파싱해 딕셔너리 리스트(level, ts, tag, user, msg)로 변환하세요.

해답:

import re

log = """[WARN] 2025-01-01 12:00:00 tag=loader user=kim msg=load fail
[INFO] 2025-01-01 12:00:05 tag=ui user=lee msg=ready"""

rx = re.compile(r"^\[(?P[A-Z]+)\]\s+(?P\d{4}-\d{2}-\d{2}\s+\d{2}:\d{2}:\d{2})\s+tag=(?P\w+)\s+user=(?P\w+)\s+msg=(?P.+)$", re.M)
rows = [m.groupdict() for m in rx.finditer(log)]
print(rows)
