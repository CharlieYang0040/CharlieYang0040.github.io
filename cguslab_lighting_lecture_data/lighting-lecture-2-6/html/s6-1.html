re 심화: 컴파일/플래그, 그룹, 고급 매칭

성능 좋고 읽기 쉬운 정규식을 위해 컴파일, 명명 그룹, non-greedy, 전방/후방 탐색, 멀티라인/도트올 플래그를 연습합니다.

[심화 문법]

  • re.compile(..., flags): I, M, S 활용
  • 명명 그룹 (?P<name>...)과 비탐욕 .*?
  • 전방/후방 탐색 (?=...), (?!...), (?<=...)
import re

text = "asset_v012.ma\nasset_v013_wip.ma\nemail: user@test.com\n\nHello\n"

# 1) 컴파일 + 명명 그룹
pat = re.compile(r"\b(?P[\w-]+)_v(?P\d{3})(?P_wip)?\.(?:ma|mb)\b", re.I)
for m in pat.finditer(text):
	print(m.groupdict())

# 2) 비탐욕 + DOTALL 로 태그 내용 추출
m = re.search(r"(.*?)", text, flags=re.S)
if m:
	print("TITLE:", m.group(1).strip())

# 3) Lookbehind 로 이메일 도메인 마스킹
masked = re.sub(r"(?<=@)[\w.-]+", "******", text)
print(masked.splitlines()[2])

[자주 하는 실수 & 팁]

탐욕성으로 인한 과다 매칭: .* 대신 .*? 사용을 고려.

# 멀티라인에서 ^/$를 라인 기준으로 쓰려면 re.M 플래그
m = re.findall(r"^[A-Z]+", "WARN\nINFO\nerror", flags=re.M|re.I)

가독성을 위해 원시 문자열 r"..."과 컴파일을 습관화.

[심화 실습]

로그에서 라인 형태 [LEVEL] YYYY-MM-DD hh:mm:ss tag=XYZ user=NAME msg=...를 모두 파싱해 딕셔너리 리스트(level, ts, tag, user, msg)로 변환하세요.

해답:

import re

log = """[WARN] 2025-01-01 12:00:00 tag=loader user=kim msg=load fail
[INFO] 2025-01-01 12:00:05 tag=ui user=lee msg=ready"""

rx = re.compile(r"^\[(?P[A-Z]+)\]\s+(?P\d{4}-\d{2}-\d{2}\s+\d{2}:\d{2}:\d{2})\s+tag=(?P\w+)\s+user=(?P\w+)\s+msg=(?P.+)$", re.M)
rows = [m.groupdict() for m in rx.finditer(log)]
print(rows)
