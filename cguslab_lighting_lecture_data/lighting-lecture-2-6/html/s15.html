mc.ls() 마스터하기: 예제 2

현재 선택한 오브젝트만 가져오기 (selection=True)

실무에서 가장 많이 사용하는 방법입니다. 아티스트가 뷰포트에서 직접 선택한 오브젝트들만 대상으로 스크립트를 실행할 때 필수적입니다. sl=True 로 줄여 쓸 수도 있어 편리합니다.

[실행 코드]

import maya.cmds as mc

# 현재 선택한 오브젝트들의 목록을 가져옵니다.
selected_objects = mc.ls(selection=True) # 또는 mc.ls(sl=True)

print("===== 현재 선택한 오브젝트 목록 =====")
# 선택한 오브젝트의 개수를 세어줍니다.
print("총 " + str(len(selected_objects)) + "개의 오브젝트가 선택되었습니다.")
for item in selected_objects:
    print(item)
print("===================================")
