3교시: 이름 규칙으로 똑똑하게 골라내 제어하기

for문 안에서 if문과 문자열 메소드를 사용해 더욱 정교하게 원하는 오브젝트만 제어합니다.

핵심 아이디어: for + if

mc.ls()로 가져온 전체 목록을 for문으로 하나씩 확인하면서, if문을 통해 특정 조건을 만족하는 오브젝트에만 setAttr 혹은 다른 명령을 실행하는 방식입니다.

더 정교한 이름 확인 방법: 문자열 메소드

단순히 이름에 특정 단어가 포함되었는지(in) 확인하는 것 외에, 파이썬이 기본으로 제공하는 유용한 문자열 메소드들을 사용할 수 있습니다.

.startswith("글자")

이름이 특정 글자로 시작하는지 확인합니다.

if object_name.startswith("Key_"):
    # 이름이 Key_로 시작하면 이 코드를 실행

.endswith("글자")

이름이 특정 글자로 끝나는지 확인합니다.

if object_name.endswith("_L"):
    # 이름이 _L로 끝나면 이 코드를 실행
