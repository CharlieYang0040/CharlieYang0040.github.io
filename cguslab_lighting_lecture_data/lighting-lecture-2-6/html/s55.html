4주차: 이제는 '툴'이다! - UI 제작과 실전 프로젝트

스크립트에 UI를 입혀 다른 사람도 쉽게 쓸 수 있는 '툴'을 만들고, 최종 포트폴리오 프로젝트를 완성합니다.

1교시: 클릭으로 스크립트 실행하기 - UI의 첫걸음

지금까지 만든 스크립트를 비전문가도 쉽게 사용할 수 있도록 '창(Window)'과 '버튼(Button)'을 만드는 방법을 배웁니다. UI의 가장 기초적인 3가지 명령어를 알아봅니다.

  • mc.window() 모든 UI 요소를 담을 '창'을 생성합니다.
  • mc.columnLayout() UI 요소들을 세로로 차곡차곡 쌓아주는 '정리함' 역할을 합니다.
  • mc.button() 클릭 가능한 '버튼'을 만들고, command 플래그로 특정 함수를 연결합니다.

2, 3교시: 최종 프로젝트 - 룩뎁 턴테이블 자동 생성기

실무에서 매우 유용한 '원클릭 룩뎁 턴테이블 자동 생성기'를 만듭니다. 3점 조명, 스튜디오 배경, 회전 카메라 애니메이션을 생성하는 모든 로직을 함수로 구현하고, 이것을 UI 버튼 하나로 실행하는 완전한 툴을 완성합니다.

이번 주 과정을 마치면 여러분의 첫 포트폴리오가 될 멋진 결과물을 얻게 될 것입니다!
