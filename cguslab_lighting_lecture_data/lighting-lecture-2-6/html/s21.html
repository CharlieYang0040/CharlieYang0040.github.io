속성 제어 예제 2: 체크박스(Boolean) 속성 제어하기 - Diffuse/Specular

라이트가 물체의 색상(Diffuse)이나 반짝임(Specular)에 영향을 줄지 결정하는 체크박스 옵션입니다.

체크박스 속성은 0 (끄기) 또는 1 (켜기)의 값을 가집니다. 이번에는 mc.ls(sl=True, lights=True) 플래그를 조합하여, 현재 선택한 오브젝트 중 라이트만 골라내어 제어하는 실용적인 예제입니다.

[실행 코드]

import maya.cmds as mc

# 현재 선택한 오브젝트 중, '라이트'에 해당하는 것들의 쉐입 노드만 가져옵니다.
selected_light_shapes = mc.ls(sl=True, lights=True)

for light_shape in selected_light_shapes:
    # 선택한 라이트가 반짝임(Specular)에 영향을 주지 않도록 설정합니다.
    mc.setAttr(light_shape + ".emitSpecular", 0) # 0은 False(끄기)와 같습니다.

print("선택한 라이트의 Specular 옵션을 껐습니다.")
결과: 뷰포트에서 선택한 라이트들의 'Emit Specular' 체크박스가 모두 해제됩니다.
