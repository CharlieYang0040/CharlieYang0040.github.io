배운 내용을 응용하여 여러 라이트의 속성을 제어하는 스크립트를 작성해보세요.

문제 1: Spot Light 속성 일괄 변경 ★★☆☆☆

Spot Light를 3개 만드세요. 모든 스포트라이트의 Cone Angle을 90으로, Penumbra Angle을 15로 한 번에 변경하는 스크립트를 작성하세요.

문제 2: 현재 값 기준으로 어둡게 만들기 ★★★☆☆

라이트를 여러 개 만들고 일부만 선택하세요. 선택한 라이트만 현재 Intensity 값에 0.5를 곱해서 어둡게 만드는 스크립트를 작성하세요. (getAttr 사용 필수)

심화 문제: 토글 기능 응용하기 ★★★★☆

'그림자 토글' 예제를 응용하여, 'Emit Diffuse 토글' 스크립트를 만들어보세요. 선택한 라이트의 emitDiffuse 속성 값을 확인하여 켜져 있으면 끄고, 꺼져 있으면 켜도록 만드세요.
