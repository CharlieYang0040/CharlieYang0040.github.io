예제 2: 버튼에 기능 연결하기 (command)

UI의 핵심! 버튼을 클릭했을 때, 미리 만들어 둔 파이썬 함수가 실행되도록 연결합니다.

[실행 코드]

import maya.cmds as mc

# 버튼을 누르면 실행될 함수를 미리 정의합니다.
# *args는 Maya가 UI에서 함수를 호출할 때 자동으로 전달하는 값을 받기 위한 관례적인 표현입니다.
def create_sphere_func(*args):
    mc.polySphere()
    print("구가 생성되었습니다!")

# UI 창 중복 생성 방지
if mc.window("functionUI", exists=True):
    mc.deleteUI("functionUI")

mc.window("functionUI", title="함수 연결 UI")
mc.columnLayout()

# command 플래그에 실행할 함수 이름을 넣어줍니다. 
# 중요: 함수 이름만 적고 괄호()는 빼야 합니다!
mc.button(label="구 생성하기", command=create_sphere_func)

mc.showWindow("functionUI")

매우 중요한 규칙

command 플래그에 함수를 연결할 때, command=create_sphere_func 와 같이 함수의 이름만 전달해야 합니다.

만약 command=create_sphere_func() 처럼 괄호를 붙이면, UI가 생성되는 그 즉시 함수가 실행되어 버리고, 정작 버튼을 클릭했을 때는 아무 일도 일어나지 않습니다.

버튼을 '클릭했을 때' 실행되게 하려면, Maya에게 '이 함수를 실행하라'는 지시서(함수의 이름)만 넘겨주는 것이라고 이해하면 쉽습니다.
