리스트 컴프리헨션

컴프리헨션은 append()를 반복하는 대신, [표현식 for 변수 in 반복가능객체 if 조건] 형태로 리스트를 한 줄에 생성하는 문법입니다.

[기본 예제]

# 1) 일반 for문 → 컴프리헨션
squares = []
for i in range(5):
    squares.append(i * i)
print(squares)

squares2 = [i * i for i in range(5)]
print(squares2)

# 2) 조건 필터 포함
odds = [n for n in range(10) if n % 2 == 1]
print(odds)

# 3) 중첩 for (간단 예)
pairs = [(x, y) for x in range(2) for y in range(3)]
print(pairs)

[실전 문제]

표준 라이브러리 random을 사용해 1~100 사이 난수 10개 리스트를 만든 뒤, 짝수만 남겨 오름차순 정렬해 출력하세요. 리스트 컴프리헨션을 반드시 사용해보세요.

해답:

import random as rnd

nums = [rnd.randint(1, 100) for _ in range(10)]
evens_sorted = sorted([n for n in nums if n % 2 == 0])
print(f"원본: {nums}")
print(f"짝수만 정렬: {evens_sorted}")
