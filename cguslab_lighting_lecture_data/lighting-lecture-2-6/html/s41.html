3교시: 함수가 결과 알려주기 - 반환(Return) 값 활용

함수가 처리한 결과 값을 돌려받아 다음 작업에 사용하는 방법을 배웁니다.

핵심 개념: 함수가 계산 결과를 돌려주는 방법

지금까지의 함수는 무언가를 '실행'만 하고 끝났습니다. 하지만 함수가 처리한 결과(예: 생성된 오브젝트의 이름, 계산된 좌표 값 등)를 돌려받아 다음 작업에 이어서 사용해야 할 때가 많습니다.
return 키워드는 두 가지 역할을 합니다:

  1. 함수의 실행을 그 자리에서 즉시 종료합니다.
  2. return 뒤에 오는 값을 함수를 호출했던 곳으로 돌려줍니다.

[Return 기본 예제]

def add(a, b):
    result = a + b
    return result # result 변수에 담긴 값을 호출한 곳으로 돌려주고 함수를 종료

# add(10, 5) 함수가 실행된 후, 그 결과인 15가 return되어 a_plus_b 변수에 저장됩니다.
a_plus_b = add(10, 5)

print(a_plus_b) # 출력: 15
