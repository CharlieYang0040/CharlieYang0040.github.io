예제 코드로 매개변수 익히기 1: 이름을 입력받아 인사하는 함수

매개변수의 가장 기본적인 활용법입니다.

greet 함수는 name이라는 매개변수(재료를 담을 그릇)를 가지고 있습니다. 함수를 호출할 때 greet("김철수")와 같이 실제 값(인자)을 넣어주면, 함수 내부의 name 변수는 "김철수"라는 값을 가지게 되어 문장을 완성합니다.

[실행 코드]

def greet(name):
    print("안녕하세요, " + name + "님! 반갑습니다.")

# 함수 호출 시 다른 '재료'를 넣어줍니다.
greet("김철수")
greet("박영희")

[실행 결과]

안녕하세요, 김철수님! 반갑습니다.
안녕하세요, 박영희님! 반갑습니다.
