오브젝트 선택 및 변형

cmds.select(), cmds.move(), cmds.rotate(), cmds.scale() 함수로 오브젝트를 제어할 수 있습니다.

[기본 예제]

import maya.cmds as cmds

my_cube_name = cmds.polyCube(name="myCube")[0]
cmds.select(my_cube_name)
cmds.move(5, 0, 0)      # X축으로 5만큼 이동
cmds.rotate(0, 45, 0)   # Y축으로 45도 회전
cmds.scale(2, 2, 2)     # 전체 크기 2배

[실전 문제]

`for` 반복문과 `cmds.polyCube()`, `cmds.move()`를 사용하여 큐브 5개를 Y축 방향으로 차곡차곡 쌓아 탑을 만들어 보세요.

해답:

import maya.cmds as cmds

for i in range(5):
    cube_name = cmds.polyCube()[0]
    # i가 0, 1, 2, 3, 4로 변하므로 큐브는 y=0, 1, 2, 3, 4 위치에 쌓입니다.
    cmds.move(0, i, 0, cube_name)
