속성 제어 예제 3: 드롭다운 메뉴(Enum) 속성 제어하기 - Decay Rate

빛이 거리에 따라 어떻게 약해질지(감쇠율)를 정하는 드롭다운 메뉴 옵션입니다.

드롭다운 메뉴(Enum 타입) 속성은 메뉴에 보이는 순서대로 0, 1, 2... 와 같은 정수(Integer) 값을 가집니다. 예를 들어, Decay Rate 속성은 'No Decay'(0), 'Linear'(1), 'Quadratic'(2), 'Cubic'(3) 순서입니다.

[실행 코드]

import maya.cmds as mc

all_light_shapes = mc.ls(lights=True)

for light_shape in all_light_shapes:
    # 모든 라이트의 감쇠율을 'Quadratic'(실세계와 가장 비슷)으로 설정합니다.
    # 'Quadratic'은 메뉴에서 3번째에 있으므로 값은 2입니다. (0부터 시작)
    mc.setAttr(light_shape + ".decayRate", 2)

print("모든 라이트의 Decay Rate를 Quadratic으로 설정했습니다.")
결과: 씬에 있는 모든 라이트의 Decay Rate가 'Quadratic'으로 변경됩니다.
