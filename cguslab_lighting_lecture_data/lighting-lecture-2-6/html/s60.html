2교시: 최종 프로젝트 1단계 - 룩뎁 턴테이블 생성 로직 구현

UI를 만들기 전에, '툴'이 수행해야 할 핵심 기능들을 먼저 함수로 구현하여 작업을 체계화합니다.

큰 그림 그리기: 기능 분리

복잡한 툴을 만들 때는 먼저 필요한 모든 기능을 잘게 쪼개어 각각 독립적인 함수로 만드는 것이 좋습니다. 이렇게 하면 코드 관리가 쉽고, 나중에 UI에 기능을 하나씩 연결하기만 하면 되므로 매우 효율적입니다.

우리가 만들 '룩뎁 턴테이블 생성기'의 핵심 기능은 아래 3가지입니다:

  1. 배경 생성 (create_backdrop)
  2. 3점 조명 생성 (create_three_point_lights)
  3. 턴테이블 카메라 생성 (create_turntable_camera)

[함수 1: 스튜디오 배경(사이클로라마) 생성]

import maya.cmds as mc

def create_backdrop(size=20):
    """
    간단한 스튜디오 배경(사이클로라마)을 생성하는 함수
    size: 배경의 크기를 결정하는 매개변수
    """
    # 1. 지정된 크기의 폴리곤 평면을 생성합니다.
    plane = mc.polyPlane(w=size, h=size, name="Backdrop_Plane")[0]
    mc.move(0, 0, -size / 2.0, plane, absolute=True)

    # 2. 평면의 뒷쪽 엣지를 선택합니다. (엣지 번호는 다를 수 있으나 보통 1)
    edge_name = f"{plane}.e[1]"
    mc.select(edge_name)
    
    # 3. 선택한 엣지를 위로 올려 벽 부분을 만듭니다.
    mc.move(0, size / 2.0, 0, r=True)

    # 4. 바닥과 벽이 만나는 부분을 부드럽게 만들기 위해 베벨을 적용합니다.
    mc.polyBevel(edge_name, fraction=0.3, segments=5)
    
    mc.select(clear=True) # 선택 해제
    print("스튜디오 배경을 생성했습니다.")
    return plane
