getAttr로 현재 값 기준 바꾸기: 토글(Toggle) 스크립트

getAttr로 현재 상태를 확인하고, if문으로 조건에 따라 다른 명령을 실행하는 스마트한 스크립트입니다.

예를 들어, 현재 그림자가 켜져 있으면 끄고, 꺼져 있으면 켜는 '토글' 기능을 만들 수 있습니다. 스크립트를 실행할 때마다 상태가 반전되어 매우 편리합니다.

[실행 코드: 그림자 토글]

import maya.cmds as mc

selected_light_shapes = mc.ls(sl=True, lights=True)

for light_shape in selected_light_shapes:
    # 현재 그림자 상태를 가져옵니다. (결과는 0 또는 1)
    is_shadow_on = mc.getAttr(light_shape + ".useDepthMapShadows")

    if is_shadow_on == 1: # 그림자가 켜져 있다면 (True)
        mc.setAttr(light_shape + ".useDepthMapShadows", 0) # 끈다 (False)
        print(light_shape + "의 그림자를 껐습니다.")
    else: # 그림자가 꺼져 있다면 (False)
        mc.setAttr(light_shape + ".useDepthMapShadows", 1) # 켠다 (True)
        print(light_shape + "의 그림자를 켰습니다.")
