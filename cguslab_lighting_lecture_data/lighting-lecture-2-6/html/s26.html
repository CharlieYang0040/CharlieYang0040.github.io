정교하게 제어하기 예제 1: .startswith()로 제어하기

이름이 Key_로 시작하는 라이트들의 강도를 일괄적으로 변경합니다.

중요 테크닉: mc.ls(lights=True)는 쉐입 노드를 반환하므로, 우리가 확인하고 싶은 트랜스폼 노드의 이름을 얻으려면 mc.listRelatives(light_shape, parent=True)[0]를 사용해야 합니다.

[실행 코드]

import maya.cmds as mc

# 예시 씬: Key_Main_Lgt, Key_Side_Lgt, Rim_Lgt 등의 라이트가 있다고 가정
all_light_shapes = mc.ls(lights=True)

for light_shape in all_light_shapes:
    # light_shape 변수는 'pointLightShape1'과 같은 형태이므로, 
    # 그것의 부모(트랜스폼 노드)의 이름을 가져와야 합니다.
    transform_node = mc.listRelatives(light_shape, parent=True)[0]

    # 트랜스폼 노드의 이름이 "Key_"로 시작하는지 확인
    if transform_node.startswith("Key_"):
        mc.setAttr(light_shape + ".intensity", 20)
        print(f"{transform_node}는 Key 라이트이므로 강도를 20으로 설정합니다.")
    else:
        print(f"{transform_node}는 Key 라이트가 아니므로 넘어갑니다.")
