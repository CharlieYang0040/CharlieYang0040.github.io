'라이트 페인터' 핵심 기능: 오브젝트 중앙에 라이트 생성

실무 툴의 핵심 원리입니다. 특정 오브젝트의 중심점을 계산해서 그 위치에 라이트를 만들어 봅시다.

mc.xform(q=True, bb=True) 명령어를 사용하면 오브젝트를 감싸는 가상의 직육면체 박스(Bounding Box)의 최소/최대 좌표를 얻을 수 있습니다. 이 값을 이용해 정확한 중심점을 계산할 수 있습니다.

[실행 코드]

import maya.cmds as mc

def create_light_at_center(target_object):
    # target_object의 바운딩 박스(Bounding Box) 좌표를 가져옵니다.
    # 결과: [xmin, ymin, zmin, xmax, ymax, zmax]
    bbox = mc.xform(target_object, q=True, bb=True, worldSpace=True)

    # 중심점 좌표를 계산합니다.
    center_x = (bbox[0] + bbox[3]) / 2.0
    center_y = (bbox[1] + bbox[4]) / 2.0
    center_z = (bbox[2] + bbox[5]) / 2.0

    # 계산된 중심점에 Point Light를 생성합니다.
    mc.pointLight(position=[center_x, center_y, center_z])
    print(f"'{target_object}'의 중심에 라이트를 생성했습니다.")

# --- 사용법 ---
