shutil 심화: copytree ignore, 아카이브, 용량

대규모 자산 백업/배포를 가정하여, 제외 패턴 복사, 압축/해제, 디스크 용량을 다룹니다.

[심화 기능]

  • copytree(..., ignore=shutil.ignore_patterns(...)): 불필요 파일 제외
  • make_archive / unpack_archive: 폴더 압축/해제
  • disk_usage: 남은 용량 체크로 사전 실패 방지
import os, shutil
from datetime import datetime

src = "textures"
clean = os.path.join("backup", "textures_clean")
os.makedirs(os.path.dirname(clean), exist_ok=True)

# 1) 제외 패턴 적용 복사
shutil.copytree(
	src, clean,
	dirs_exist_ok=True,
	ignore=shutil.ignore_patterns("*.tmp", "*.bak", "Thumbs.db", "__pycache__")
)

# 2) 아카이브 생성 (zip)
stamp = datetime.now().strftime("%Y%m%d")
archive_base = os.path.join("backup", f"textures_{stamp}")
archive_path = shutil.make_archive(archive_base, "zip", root_dir="backup", base_dir="textures_clean")
print("아카이브:", archive_path)

# 3) 용량 확인
usage = shutil.disk_usage(os.getcwd())
print("남은 용량(GB):", round(usage.free/1024**3, 2))

[자주 하는 실수 & 성능 팁]

copytree는 기본적으로 대상이 비어있어야 합니다. 3.8+에서는 dirs_exist_ok=True로 덮어쓰기를 허용할 수 있습니다.

[심화 실습]

현재 폴더의 assets를 제외 패턴(*.tmp, __pycache__)로 복사 후, backup/assets_YYYYMMDD.zip으로 압축하고, 임시 폴더에 해제하여 파일 수가 원본과 동일한지 검증하세요.

해답:

import os, shutil
from datetime import datetime

src = "assets"
clean = os.path.join("backup", "assets_clean")
os.makedirs(os.path.dirname(clean), exist_ok=True)
shutil.copytree(src, clean, dirs_exist_ok=True, ignore=shutil.ignore_patterns("*.tmp", "__pycache__"))

stamp = datetime.now().strftime("%Y%m%d")
archive = shutil.make_archive(os.path.join("backup", f"assets_{stamp}"), "zip", root_dir="backup", base_dir="assets_clean")

# 검증: 임시 해제 후 파일 수 비교
tmp = os.path.join("backup", "_verify")
if os.path.exists(tmp):
	shutil.rmtree(tmp)
shutil.unpack_archive(archive, tmp)

# 파일 수 세기
def count_files(d):
	n = 0
	for _, _, files in os.walk(d):
		n += len(files)
	return n

print("원본:", count_files(clean), "해제본:", count_files(tmp))
