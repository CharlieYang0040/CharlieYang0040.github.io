예제 코드로 함수 익히기 1: 가장 간단한 함수

자주 사용하는 간단한 출력 코드를 함수로 만들어 재사용해 봅시다.

코드의 중간중간에 시각적인 구분선을 넣고 싶을 때, print("=" * 40) 코드를 반복해서 사용해야 합니다. 이것을 print_separator() 라는 이름의 함수로 만들어두면, 필요할 때마다 함수의 이름만 호출하여 코드를 간결하게 유지할 수 있습니다.

[실행 코드]

import maya.cmds as mc

# '구분선을 출력한다'는 기능을 가진 함수 정의
def print_separator():
    print("=" * 40)

# 함수 호출(실행)
print_separator()
print("첫번째 작업 완료")
print_separator()
print("두번째 작업 완료")
print_separator()
