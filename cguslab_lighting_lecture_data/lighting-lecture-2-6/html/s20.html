속성 제어 예제 1: 숫자(Float) 속성 제어하기 - Intensity

가장 기본적인 숫자 값(소수점이 있는 Float 타입) 변경 예제입니다.

mc.ls(lights=True)로 씬의 모든 라이트 쉐입을 가져온 뒤, for문을 돌면서 각 라이트의 .intensity 속성을 setAttr로 변경합니다.

[실행 코드]

import maya.cmds as mc

all_light_shapes = mc.ls(lights=True)

for light_shape in all_light_shapes:
    # 모든 라이트의 Intensity 값을 10으로 고정합니다.
    # "오브젝트명.속성명" 형태로 문자열을 합쳐줍니다.
    mc.setAttr(light_shape + ".intensity", 10)

print("모든 라이트의 강도를 10으로 설정했습니다.")
결과: 씬에 있는 모든 라이트의 Intensity 채널 박스 값이 10으로 변경됩니다.
