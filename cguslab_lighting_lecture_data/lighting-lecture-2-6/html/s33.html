예제 코드로 함수 익히기 2: 씬 정리 함수 1 - Freeze Transformations

모델링이나 리깅에서 정말 많이 사용하는 기능을 함수로 만들어 봅시다.

선택한 오브젝트의 위치, 회전, 크기 값을 기본값(0, 0, 0 / 1, 1, 1)으로 초기화하는 기능입니다. 특히, 아무것도 선택하지 않고 실행하는 경우를 대비하여 경고 메시지를 띄우고 함수를 안전하게 종료시키는 예외 처리를 추가합니다.

[실행 코드]

import maya.cmds as mc

def freeze_selected_object():
    # 현재 선택한 오브젝트 목록을 가져옵니다.
    selection = mc.ls(sl=True)
    if not selection: # 만약 선택한 것이 없다면 (리스트가 비어있다면)
        print("오브젝트를 먼저 선택해주세요.")
        return # 함수를 즉시 종료합니다.

    # 선택한 오브젝트의 Transform 값을 초기화합니다.
    mc.makeIdentity(selection, apply=True, t=1, r=1, s=1, n=0)
    print(str(selection) + "의 Transform 값을 초기화했습니다.")
