함수 3: 턴테이블 카메라 및 애니메이션 생성

오브젝트를 바라보며 120프레임 동안 360도 회전하는 카메라 애니메이션을 코드로 생성합니다.

[함수 3: 턴테이블 카메라 생성]

import maya.cmds as mc

def create_turntable_camera(target_object):
    """
    오브젝트를 바라보며 회전하는 카메라와 애니메이션을 생성하는 함수
    """
    # 1. 카메라를 생성하고, 이름을 지정합니다.
    cam_nodes = mc.camera(name="Turntable_Cam")
    cam_transform = cam_nodes[0]

    # 2. 생성된 카메라가 target_object를 꽉 채워 보이도록 뷰를 맞춥니다.
    mc.viewFit(cam_transform, target_object)
    
    # 3. 카메라를 뒤로 조금 물러나게 하여 오브젝트와 거리를 둡니다.
    mc.move(0, 0, 30, cam_transform, relative=True, objectSpace=True)

    # 4. 카메라를 그룹으로 묶어 회전의 중심축(Pivot)을 월드 중심으로 만듭니다.
    #    이렇게 해야 카메라가 제자리에서 돌지 않고, 원점을 중심으로 공전합니다.
    cam_group = mc.group(cam_transform, name="Turntable_Cam_Group")

    # 5. 애니메이션 키를 설정합니다. (1프레임과 120프레임)
    mc.setKeyframe(cam_group, attribute='rotateY', time=1, value=0)
    mc.setKeyframe(cam_group, attribute='rotateY', time=120, value=360)

    # 6. 애니메이션의 전체 재생 시간을 1~120 프레임으로 설정합니다.
    mc.playbackOptions(minTime=1, maxTime=120)

    print("턴테이블 카메라를 생성했습니다.")
    return cam_group
