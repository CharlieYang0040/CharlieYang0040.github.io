예제 1: 가장 기본적인 UI 창 띄우기

UI를 만드는 4단계 순서를 익히고, 오류 없이 창을 띄우는 방법을 알아봅니다.

UI 제작은 항상 정해진 순서대로 진행됩니다.

  1. mc.window(): 창을 만든다.
  2. mc.columnLayout(): UI 요소들을 담을 레이아웃을 짠다.
  3. mc.button(): 필요한 UI 요소(버튼 등)를 넣는다.
  4. mc.showWindow(): 창을 화면에 보여준다.

실무 Tip!

코드를 수정하고 다시 실행할 때 오류가 나는 것을 막기 위해, 창을 만들기 전에 if mc.window(..., exists=True): 조건문으로 이미 같은 이름의 창이 있는지 확인하고 있다면 mc.deleteUI()로 먼저 닫아주는 것이 좋습니다.

[실행 코드]

import maya.cmds as mc

# 만약 이미 같은 이름의 창이 있다면 삭제 (오류 방지)
if mc.window("myFirstUI", exists=True):
    mc.deleteUI("myFirstUI")

# 1. "myFirstUI" 라는 이름의 창을 만듭니다.
mc.window("myFirstUI", title="나의 첫 UI", widthHeight=(200, 100))

# 2. UI 요소들을 세로로 쌓을 레이아웃을 만듭니다.
mc.columnLayout()

# 3. 버튼을 만듭니다. label은 버튼에 표시될 글자입니다.
mc.button(label="여기를 누르세요")

# 4. 창을 화면에 보여줍니다.
mc.showWindow("myFirstUI")
