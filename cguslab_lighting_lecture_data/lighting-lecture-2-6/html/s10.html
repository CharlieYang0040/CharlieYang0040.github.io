채널 박스 속성 제어

cmds.setAttr()과 cmds.getAttr()을 사용하면 채널 박스의 모든 속성(Attribute) 값을 직접 설정하거나 가져올 수 있습니다.

[기본 예제]

import maya.cmds as cmds

sphere_name = cmds.polySphere(name="mySphere")[0]

# setAttr로 Y축 위치(translateY)를 3으로 설정
cmds.setAttr(f"{sphere_name}.translateY", 3)

# getAttr로 X축 크기(scaleX) 값을 가져와 출력
scale_x = cmds.getAttr(f"{sphere_name}.scaleX")
print(f"스케일 X 값: {scale_x}")

[실전 문제]

실린더와 큐브를 만들고, `setAttr`을 사용하여 큐브가 실린더의 꼭대기 중앙에 오도록 정확하게 배치하세요. (실린더 높이 5로 설정)

해답:

import maya.cmds as cmds

pillar_info = cmds.polyCylinder(name="Pillar", ch=True)
pillar_node = pillar_info[1]
light_name = cmds.polyCube(name="Light")[0]

# 실린더 높이를 5로 설정
cmds.setAttr(f"{pillar_node}.height", 5)

# 높이가 5인 실린더의 꼭대기 Y좌표는 2.5
cmds.setAttr(f"{light_name}.translateY", 2.5)
