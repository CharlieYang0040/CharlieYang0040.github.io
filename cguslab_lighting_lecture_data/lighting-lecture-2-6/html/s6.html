RE

정규표현식 re

문자열에서 패턴을 찾고, 추출하고, 치환하는 강력한 도구입니다. findall, search, sub를 우선 익혀두세요.

심화 슬라이드 보기정규표현식 따라하기

[기본 예제]

import re

text = "이메일: user01@test.com, backup: admin+cg@domain.co.kr"

# 이메일 모두 찾기
emails = re.findall(r"[\w.+-]+@[\w.-]+\.[A-Za-z]{2,}", text)
print(emails)

# 첫 번째 이메일만 찾기
m = re.search(r"([\w.+-]+)@([\w.-]+)", text)
if m:
    print("유저:", m.group(1), "도메인:", m.group(2))

# 도메인 마스킹 치환
masked = re.sub(r"(@)[\w.-]+", r"@******", text)
print(masked)

[실전 문제]

문자열에서 Maya 씬 파일명 패턴 something_v###.ma 또는 .mb를 모두 찾아 리스트로 출력하세요. (예: asset_v012.ma)

해답:

import re

data = "asset_v012.ma prop_v003.mb shotA_v120.ma temp.txt"
pattern = r"\b[\w-]+_v\d{3}\.(?:ma|mb)\b"
print(re.findall(pattern, data))
