Python 모듈과 import

모듈은 파이썬 코드(함수, 변수, 클래스)를 담은 파일입니다. 다른 파일에서 import로 불러와 재사용합니다. 표준 라이브러리(예: math, random)를 먼저 연습한 뒤 Maya의 maya.cmds를 import할 예정입니다.

[기본 예제]

# 1) 모듈 전체 import
import math
print(math.pi)
print(math.sqrt(16))

# 2) 별칭(alias) 사용
import math as m
print(m.sin(m.pi / 2))

# 3) 필요한 심볼만 가져오기
from random import randint, choice
print(randint(1, 10), choice(['A', 'B', 'C']))

[실전 문제]

표준 라이브러리 random 모듈을 import한 뒤, 1~6 사이의 정수를 3번 무작위로 뽑아 리스트로 출력하세요. 가능하면 as 별칭을 사용해보세요. 출력 형식 예: 주사위: [3, 5, 1]

해답:

import random as rnd

rolls = [rnd.randint(1, 6) for _ in range(3)]
print(f"주사위: {rolls}")
