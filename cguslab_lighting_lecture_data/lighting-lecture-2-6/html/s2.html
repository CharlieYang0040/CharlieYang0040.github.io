Maya 스크립트 에디터

Maya에서 코드를 작성하고 실행하는 공간입니다. `Python` 탭을 사용하며, 스크립트 시작 시 import maya.cmds as cmds 코드로 Maya의 명령어 모음을 불러와야 합니다.

[기본 예제]

import maya.cmds as cmds

# 스크립트 에디터의 결과창에 메시지를 출력합니다.
print("Hello, Maya!")

[실전 문제]

스크립트 에디터에서 `import maya.cmds as cmds`를 먼저 작성한 뒤, 자신의 이름을 `print()` 함수로 출력하는 코드를 작성하고 실행해보세요.

해답:

import maya.cmds as cmds

my_name = "김마야"
print(f"제 이름은 {my_name}입니다. Maya와 친구가 되고 싶어요!")
