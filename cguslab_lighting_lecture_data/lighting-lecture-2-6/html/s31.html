개념 설명: 왜 함수를 써야 할까? (코드의 재활용)

반복을 피하고, 코드를 효율적으로 관리하기 위한 첫걸음입니다.

불편한 점: 복사 & 붙여넣기

똑같은 코드를 여러 번 복사해서 사용하면, 나중에 수정할 일이 생겼을 때 모든 곳을 일일이 찾아다니며 똑같이 바꿔줘야 합니다. 실수하기도 쉽고 매우 번거롭습니다.

해결책: 함수(Function)

함수는 코드 묶음에 이름을 붙여주는 것과 같습니다. 우리는 그 이름만 부르면(호출하면) 언제든 해당 코드 묶음을 실행할 수 있습니다. 마치 게임의 '스킬 단축키'와 같습니다.

[기본적인 함수 정의(Define)와 호출(Call) 방법]

# 'def' 키워드로 'my_function'이라는 이름의 함수를 정의합니다.
def my_function():
    # 이 함수가 실행할 코드를 들여쓰기하여 작성합니다.
    print("안녕하세요!")
    print("함수가 실행되었습니다.")

# 이제 이름만 부르면 함수 안의 코드가 실행됩니다.
my_function() # 함수 호출
my_function() # 여러 번 재사용 가능
