RE

re 따라하기: 기본 패턴에서 조합까지

설명 → 바로 실습 → 즉시 정답 확인 흐름으로, 여러 정규표현식을 손에 익힙니다.

[이메일 매칭]

요구사항: 사용자명(문자/숫자/._+-), 도메인, 최종 TLD 2자 이상

import re

text = "이메일: user01@test.com, backup: admin+cg@domain.co.kr"
pattern = r"[\w.+-]+@[\w.-]+\.[A-Za-z]{2,}"
print(re.findall(pattern, text))

[버전 태그 v###]

요구사항: v 다음 세 자리 숫자

import re

text = "asset_v012.ma prop_v3.mb prop_v003.mb"
print(re.findall(r"\bv\d{3}\b", text))

[전화번호 한국식]

요구사항: 02 또는 0XX-XXX(X)-XXXX

import re

text = "전화: 010-1234-5678, 예비: 02-987-6543, 오류: 010-12-5678"
print(re.findall(r"\b\d{2,3}-\d{3,4}-\d{4}\b", text))

[파일명 패턴 something_v###.(ma|mb)]

요구사항: 하이픈/밑줄 포함 이름, 확장자 ma 또는 mb

import re

data = "asset_v012.ma prop-v003.mb shotA_v120.ma temp.txt"
pattern = r"\b[\w-]+_v\d{3}\.(?:ma|mb)\b"
print(re.findall(pattern, data))

[태그 안의 내용 비탐욕 추출]

요구사항: ... 사이 텍스트 (줄바꿈 포함)

import re

html = "\nHello\n"
m = re.search(r"(.*?)", html, flags=re.S)
print(m.group(1).strip() if m else None)

[라인별 로그 파싱]

요구사항: [LEVEL] 2025-01-01 12:00:00 msg=...

import re

log = """[WARN] 2025-01-01 12:00:00 msg=load fail
[INFO] 2025-01-01 12:00:05 msg=ready"""
rx = re.compile(r"^\[(?P[A-Z]+)\]\s+(?P\d{4}-\d{2}-\d{2}\s+\d{2}:\d{2}:\d{2})\s+msg=(?P.+)$", re.M)
print([m.groupdict() for m in rx.finditer(log)])
