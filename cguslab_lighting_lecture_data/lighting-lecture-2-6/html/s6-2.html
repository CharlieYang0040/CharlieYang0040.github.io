정규표셔식 심화: 자주 쓰는 기호 사전

자주 쓰는 정규식 기호를 한 눈에. 간단 예시와 함께 빠르게 찾아보세요.

[앵커(Anchors) · 경계]

  • ^: 문자열/라인 시작, $: 끝
  • \b: 단어 경계, \B: 비-단어 경계
import re
print(bool(re.search(r"^hello$", "hello")))  # True
print(re.findall(r"\bcat\b", "copy cat, category"))  # ['cat']

[문자 · 글자 클래스]

  • .: 임의 문자(줄바꿈 제외)
  • [abc]: a/b/c 중 하나, [^abc]: a/b/c 제외
  • [a-zA-Z]: 범위, \d/\w/\s
import re
print(re.findall(r"gr.y", "grey gray grow"))  # ['grey', 'gray']
print(re.findall(r"[0-9][A-F]", "3A 7G 9F"))  # ['3A', '9F']

[수량자(Quantifiers)]

  • *: 0+회, +: 1+회, ?: 0/1회
  • {m}, {m,n}: 횟수 지정
  • 게으른 수량자: 뒤에 ? 추가
import re
html = "hithere"
print(re.findall(r".*?", html))  # ['hi', 'there']

[그룹과 캡처(Grouping and Capturing)]

(...): 캡처 그룹. 괄호 안의 패턴과 일치하는 부분을 캡처합니다.

re.search(r'(...)-(...)', 'apple-banana').groups()  # ('apple', 'banana')

(?P<name>...): 이름있는 캡처 그룹. 그룹에 이름을 지정합니다.

m = re.search(r'Hello (?P<name>\\w+)', 'Hello Alice')
print(m.group('name')) # 'Alice'

(?:...): 비캡처 그룹. 그룹화는 하지만 캡처는 하지 않습니다.

re.findall(r'(?:first|second) name is (\\w+)', 'first name is Alice, second name is Bob') # ['Alice', 'Bob']

(?P=name): 이름있는 그룹을 참조합니다.

re.search(r'(\\w+) and \\1', 'word and word') is not None # True

\\number: 캡처된 그룹을 번호로 다시 참조합니다.

m = re.search(r'(?P<word>\\w+) and (?P=word)', 'word and word')
print(m is not None) # True
