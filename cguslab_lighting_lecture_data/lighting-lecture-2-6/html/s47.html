예제: V-Ray 재질 생성 및 속성 설정

V-Ray 재질도 원리는 같습니다. 노드 타입만 'VRayMtl'로 바꿔주면 됩니다.

V-Ray와 같은 외부 렌더러의 재질을 생성하기 위해서는 해당 플러그인이 Maya에 로드되어 있어야 합니다. 스크립트 시작 부분에서 플러그인 로드 여부를 확인하고, 로드되어 있지 않다면 자동으로 로드하는 코드를 추가하여 안정성을 높일 수 있습니다.

[실행 코드: V-Ray 크롬 재질 만들기]

import maya.cmds as mc

# V-Ray 플러그인이 로드되었는지 확인하고, 안되어있으면 로드합니다.
if not mc.pluginInfo('vrayformaya', query=True, loaded=True):
    mc.loadPlugin('vrayformaya')

# 'chrome_vray_mat' 이름의 VRayMtl 재질 생성
chrome_mat = mc.shadingNode('VRayMtl', asShader=True, name='chrome_vray_mat')

# Diffuse 색상을 어둡게 설정
mc.setAttr(chrome_mat + '.color', 0.1, 0.1, 0.1, type='double3')

# Reflection 색상을 밝게 하여 반사도를 높임
mc.setAttr(chrome_mat + '.reflectionColor', 1, 1, 1, type='double3')

# Reflection Glossiness를 조절하여 반사의 선명도를 제어 (0.95 = 약간 번진 반사)
mc.setAttr(chrome_mat + '.reflectionGlossiness', 0.95)

print(f"'{chrome_mat}' V-Ray 재질을 생성하고 속성을 설정했습니다.")
