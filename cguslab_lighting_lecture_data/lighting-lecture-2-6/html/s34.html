예제 코드로 함수 익히기 3: 씬 정리 함수 2 - Delete History

오브젝트에 쌓인 불필요한 작업 내역(히스토리)을 지우는 기능입니다.

씬을 깔끔하게 유지하고 예기치 않은 오류를 방지하기 위해 주기적으로 히스토리를 삭제하는 것은 좋은 습관입니다. 이 반복적인 작업을 함수로 만들어두면 매우 편리합니다.

[실행 코드]

import maya.cmds as mc

def delete_history_on_selected():
    selection = mc.ls(sl=True)
    if not selection:
        print("오브젝트를 먼저 선택해주세요.")
        return

    # 선택한 오브젝트의 히스토리를 삭제합니다.
    mc.delete(selection, constructionHistory=True)
    print(str(selection) + "의 히스토리를 삭제했습니다.")

# --- 사용법 ---
# 베벨(Bevel)이나 스무스(Smooth)를 적용한 오브젝트를 선택하고 아래 함수를 호출해보세요.
