3교시: 최종 프로젝트 2단계 - UI 통합 및 툴 완성

지금까지 만든 모든 기능 함수들을 하나의 '마스터 함수'로 묶고, 최종 UI 버튼에 연결하여 툴을 완성합니다.

1단계: 마스터 함수 제작

UI 버튼에 함수를 하나만 연결할 수 있으므로, 2교시에서 만든 3개의 핵심 기능 함수들을 순서대로 호출하는 '마스터 함수'를 만듭니다.

이 함수는 전체 작업의 흐름을 제어하는 '컨트롤 타워' 역할을 합니다. 또한, 사용자가 오브젝트를 선택하지 않고 버튼을 누르는 실수를 방지하기 위한 간단한 오류 처리(mc.warning)도 포함합니다.

[마스터 함수: run_turntable_generator]

import maya.cmds as mc

# (2교시에서 만든 3개의 함수: create_backdrop, create_three_point_lights,
#  create_turntable_camera가 이 코드 위에 있다고 가정합니다)

def run_turntable_generator(*args):
    # 1. 현재 선택한 오브젝트를 확인합니다.
    selection = mc.ls(sl=True)
    if not selection:
        mc.warning("오브젝트를 먼저 선택해주세요!") # 경고 메시지 출력
        return # 함수 종료

    target = selection[0]

    # 2. 핵심 기능 함수들을 순서대로 호출합니다.
    create_backdrop()
    create_three_point_lights(target)
    create_turntable_camera(target)

    # 3. 사용자 편의를 위해 마지막에 오브젝트를 다시 선택해줍니다.
    mc.select(target)
    print("\n===== 턴테이블 씬 생성이 완료되었습니다! =====")
