정교하게 제어하기 예제 2: 트랜스폼 노드 제어하기

지금까지는 라이트의 색, 강도 등 '쉐입' 노드의 속성을 바꿨습니다. 이번엔 '트랜스폼' 노드를 직접 제어하여 위치를 바꿔봅시다.

mc.ls(type='light')를 사용하면 라이트의 '트랜스폼' 노드 목록을 직접 얻을 수 있습니다. 이 목록을 이용해 특정 이름 규칙을 가진 라이트 그룹 전체를 이동시키는 등의 작업이 가능합니다.

[실행 코드: 특정 라이트 그룹 옮기기]

import maya.cmds as mc

# 예시 씬: Front_Lgt_L, Rim_Lgt_L, Front_Lgt_R 등의 라이트가 있다고 가정
# type='light'는 light의 트랜스폼 노드를 직접 가져옵니다.
all_light_transforms = mc.ls(type='light') 

for light in all_light_transforms:
    # 이름이 '_L' (왼쪽)으로 끝나는지 확인
    if light.endswith("_L"): 
        # 현재 x 위치를 가져와서 -5만큼 더 왼쪽으로 옮깁니다.
        current_tx = mc.getAttr(light + ".translateX")
        mc.setAttr(light + ".translateX", current_tx - 5)
        print(f"{light}를 왼쪽으로 5만큼 더 이동시켰습니다.")
