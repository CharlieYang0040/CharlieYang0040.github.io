SH

파일 조작 shutil

shutil은 파일/디렉터리 복사, 이동, 삭제, 압축 등 고수준 파일 작업을 제공합니다.

심화 슬라이드 보기

[기본 예제]

import os
import shutil

src = os.path.join(os.getcwd(), "example.txt")
dst = os.path.join(os.getcwd(), "backup", "example.txt")

# 디렉터리 준비
os.makedirs(os.path.dirname(dst), exist_ok=True)

# 파일 복사
shutil.copy2(src, dst)  # 메타데이터 포함

# 파일 이동 (이름 변경 포함)
new_dst = dst.replace("example", "example_v2")
shutil.move(dst, new_dst)

# 디렉터리 통째로 복사
shutil.copytree("assets", "assets_copy", dirs_exist_ok=True)

# 디렉터리 삭제
shutil.rmtree("assets_copy")

[실전 문제]

현재 폴더의 textures 디렉터리를 backup/textures_YYYYMMDD로 백업하세요. (힌트: os.makedirs, shutil.copytree)

해답:

import os
import shutil
from datetime import datetime

src = "textures"
stamp = datetime.now().strftime("%Y%m%d")
dst = os.path.join("backup", f"textures_{stamp}")

os.makedirs(os.path.dirname(dst), exist_ok=True)
shutil.copytree(src, dst, dirs_exist_ok=True)
print("백업 완료:", dst)
