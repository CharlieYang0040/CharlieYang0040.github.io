변수와 자료형

변수는 데이터를 저장하는 '이름표가 붙은 상자'이며, 자료형은 변수에 담을 수 있는 데이터의 종류(정수, 실수, 문자열 등)를 말합니다.

[기본 예제]

# 다양한 자료형의 데이터를 변수에 저장
level = 10         # 정수 (Integer)
exp = 85.5         # 실수 (Float)
job = "마법사"      # 문자열 (String)
is_alive = True    # 불리언 (Boolean)

# 변수에 저장된 값들을 화면에 출력
print("레벨:", level)
print("경험치:", exp)
print("직업:", job)
print("생존 여부:", is_alive)

[실전 문제]

자신의 이름, 나이, 사는 도시를 각각 `name`, `age`, `city` 라는 변수에 알맞은 자료형으로 저장해 보세요.

해답:

name = "홍길동"
age = 25
city = "서울"

print(name, age, city)
