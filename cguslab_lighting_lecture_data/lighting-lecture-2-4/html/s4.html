마법의 연결고리: API와 모듈

API (Application Programming Interface)

프로그램이 제공하는 기능 목록과 사용 설명서입니다. 레스토랑의 '메뉴판'과 같습니다. 우리는 메뉴판(API)을 보고 요리(기능)를 주문(코드 실행)합니다.

모듈 (Module)

API를 담고 있는 파이썬 파일(.py) 또는 파일 묶음입니다. Maya에서는 maya.cmds가 가장 대표적인 모듈(메뉴판)입니다.

[코드 분석]

import maya.cmds as cmds
  • import maya.cmds: "Maya의 cmds 라는 모듈(메뉴판)을 사용하겠습니다."
  • as cmds: "앞으로는 maya.cmds 라는 긴 이름 대신 cmds 라는 별명으로 부르겠습니다."
