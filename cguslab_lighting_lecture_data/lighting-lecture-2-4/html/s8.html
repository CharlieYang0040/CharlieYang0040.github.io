함수 (Function)

특정 작업을 수행하는 코드 덩어리에 이름을 붙인 것입니다. 코드를 재사용하고 프로그램을 체계적으로 만드는 데 필수적입니다.

[기본 예제]

# 두 숫자를 더하는 함수를 정의(define)
def add(a, b):
    result = a + b
    return result # 결과를 반환

# 함수를 호출(call)하여 사용
sum_value = add(5, 10)
print(sum_value) # 15가 출력됨

[실전 문제]

숫자 두 개(a, b)를 매개변수로 받아서 `a - b`의 결과를 반환(`return`)하는 `subtract` 라는 이름의 함수를 만들어보세요.

해답:

def subtract(a, b):
    return a - b

result = subtract(100, 50)
print(result) # 50이 출력됨
