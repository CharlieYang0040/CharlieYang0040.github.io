Sky/Sun 분리 기법: 실습 단계

두 개의 DomeLight 역할 분담하기

🌌 Dome 1: 하늘 (Sky/Ambient)
  • Color Map: 원본 HDRI 연결
  • 역할: 장면의 전반적인 분위기, 색감, 부드러운 반사
  • 제어: 우선 기본값으로 두고 Sun Light 추가 후 미세 조정
☀️ Dome 2: 태양 (Sun/Key Source)
  • Color Map: 태양만 남긴 HDRI 연결
  • 역할: 강한 하이라이트, 선명한 그림자 생성
  • 제어: Intensity / Exposure로 그림자 선명도 조절
  • 핵심: Rotate Y로 그림자 방향 자유롭게 제어
