4. 빛을 디자인하다: Light Filters

조명에 창의적인 효과를 더하는 강력한 기능

🚪 PxrBarnLightFilter (반 도어)

스튜디오의 '반 도어'처럼, 빛이 퍼지는 영역을 네 개의 문(Edge)으로 제어합니다.

  • 용도: 특정 영역에만 빛을 비추거나 차단
  • 예시: 사각형 창문으로 들어오는 듯한 빛 연출
🖼️ PxrGoboLightFilter (고보)

텍스처(파일)를 이용해 빛에 특정 무늬를 입힙니다.

  • 용도: 빛으로 극적인 패턴이나 그림자 생성
  • 예시: 창문 틈, 나뭇잎 사이로 비치는 그림자 연출
