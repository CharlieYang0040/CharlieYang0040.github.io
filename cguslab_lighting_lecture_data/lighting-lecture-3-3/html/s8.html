3. 3점 조명: 빛으로 형태를 조각하다

피사체를 돋보이게 만드는 핵심 조명 설계

🔦 Key Light (주 조명)

장면에서 가장 강한 주된 광원. 피사체의 형태와 질감을 결정합니다.

  • 위치: 보통 45° 위, 45° 옆
  • 라이트: PxrRectLight, PxrDiskLight
  • 제어: 크기(Size)로 그림자 부드러움 조절
💡 Fill Light (보조 조명)

Key Light가 만든 강한 그림자를 부드럽게 채워주는 빛.

  • 위치: Key Light 반대편
  • 밝기: Key Light보다 훨씬 약하게
  • 역할: 장면의 대비(Contrast) 조절
✨ Rim Light (윤곽 조명)

피사체 뒤에서 비춰 배경과 분리시키고 실루엣을 강조합니다.

  • 역할: 입체감, 깊이감 부여
  • 핵심: 정밀한 위치/각도 조절이 중요
