PxrDomeLight 기본 사용법

간단한 3단계로 사실적인 환경광 만들기

⚙️ 기본 절차
  1. 생성 및 연결:
    • RenderMan 셸프에서 PxrDomeLight 생성
    • Attribute Editor의 Color Map에 HDRI 파일 연결
  2. 밝기 조절:
    • Intensity: 전체적인 밝기 조절
    • Exposure: 스톱(Stop) 단위로 정밀 조절
  3. 방향 조절:
    • Transform 노드에서 Rotate Y 값으로 회전

Attribute Editor에서 Color Map, Intensity, Exposure 등을 설정합니다.
