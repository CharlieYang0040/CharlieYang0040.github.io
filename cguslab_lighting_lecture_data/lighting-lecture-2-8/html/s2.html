라이팅 매니저 툴 만들기

실제 툴은 사용자가 텍스트나 숫자를 입력하는 등 더 다양한 상호작용이 필요합니다. floatSliderGrp, colorSliderGrp 등을 사용해 사용자 입력을 받을 수 있습니다.

[실전 문제 11-1]

3단계의 라이트 생성 로직을 기반으로, 라이트의 Intensity와 Color를 UI 슬라이더로 제어할 수 있는 '라이팅 매니저' 툴을 만들어보세요.

  • '라이트 생성' 버튼으로 선택한 오브젝트에 라이트를 추가합니다.
  • Intensity, Color 슬라이더를 조작하면 선택된 모든 라이트의 속성이 한번에 실시간으로 바뀌어야 합니다. (힌트: dragCommand 사용)

해답:

import maya.cmds as cmds

# --- 로직 함수들 ---
def create_lights_on_selection(*args): # ... 라이트 생성 로직 ...
def control_selected_lights_intensity(*args):
    intensity_value = cmds.floatSliderGrp(intensity_slider, q=True, v=True)
    for light in cmds.ls(sl=True, type='light'):
        cmds.setAttr(f"{light}.intensity", intensity_value)
def control_selected_lights_color(*args):
    color_value = cmds.colorSliderGrp(color_picker, q=True, rgbValue=True)
    for light in cmds.ls(sl=True, type='light'):
        cmds.setAttr(f"{light}.color", *color_value, type='double3')

# --- UI 부분 ---
window_name = "LightingManagerTool"
if cmds.window(window_name, exists=True): cmds.deleteUI(window_name)
cmds.window(window_name, title="라이팅 매니저")
cmds.columnLayout(adj=True, rowSpacing=10, m=10)
cmds.button(label="선택한 오브젝트에 라이트 생성", command=create_lights_on_selection)
cmds.separator(style='in', h=20)
cmds.text(label="선택한 라이트 제어", font="boldLabelFont")
intensity_slider = cmds.floatSliderGrp(label="Intensity", field=True, v=1.0, dragCommand=control_selected_lights_intensity)
color_picker = cmds.colorSliderGrp(label="Color", rgb=(1, 1, 1), dragCommand=control_selected_lights_color)
cmds.showWindow(window_name)
