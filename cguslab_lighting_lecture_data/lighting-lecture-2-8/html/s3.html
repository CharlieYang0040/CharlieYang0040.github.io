스크립트 모듈화와 경로

코드가 길어지면 기능별로 별도의 .py 파일(모듈)로 나누어 관리합니다. Maya는 내 문서/maya/scripts 폴더에 있는 스크립트를 import 할 수 있습니다.

[기본 예제]

# my_utils.py 파일에 함수를 저장
def say_hello():
    print("Hello from module!")

# 스크립트 에디터에서 실행
import my_utils
from importlib import reload
reload(my_utils) # 수정사항 적용

my_utils.say_hello()
