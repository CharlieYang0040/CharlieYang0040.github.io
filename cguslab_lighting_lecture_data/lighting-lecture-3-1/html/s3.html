Path Tracing: 빛의 경로를 역추적하는 탐험가

하나의 픽셀이 완성되기까지의 여정

🗺️ 원리: 가상의 탐험가
  1. 카메라(눈)의 한 픽셀에서 가상의 탐험가(광선)를 장면으로 발사합니다.
  2. 탐험가는 물체에 부딪히며 색상, 재질 정보를 기록합니다.
  3. 물리 법칙에 따라 튕겨나가기를 반복, 광원을 찾을 때까지 경로를 추적합니다.
  4. 탐험가가 지나온 모든 경로(Path)의 정보를 종합하여 최종 픽셀 색상을 계산합니다.

하나의 픽셀을 위해 수많은 '탐험가'가 각기 다른 경로를 탐험합니다.

✨ 결과와 노이즈
  • 수백, 수천 개의 경로 정보가 모여 사실적인 간접광, 부드러운 그림자, 자연스러운 반사/굴절을 생성합니다.
  • 이미지의 노이즈는 '탐험가'의 수가 부족하여 충분한 정보를 모으지 못했을 때 발생하는 '불확실성'의 결과입니다.
