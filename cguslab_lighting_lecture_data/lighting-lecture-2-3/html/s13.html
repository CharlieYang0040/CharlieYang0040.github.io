4. Variants: 버전 관리하기

하나의 Prim 안에 여러 버전의 데이터 세트(Variant Set)를 만들어두고, 필요에 따라 선택하여 보여주는 기능입니다.

① sword_asset.usda

#usda 1.0
def "Sword" (
    variantSets = "damageState"
    variants = {
        string damageState = "clean"
    }
)
{
    variantSet "damageState" = {
        "clean" {
            # 깨끗한 버전의 모델, 재질 등 정의
            string condition = "new"
        }
        "damaged" {
            # 손상된 버전의 모델, 재질 등 정의
            string condition = "broken"
        }
    }
}

② scene.usda

#usda 1.0
def "Scene"
{
    def "HeroSword" (
        references = @./sword_asset.usda@
    )
    {
        # variant 선택을 덮어씀
        variants = {
            string damageState = "damaged"
        }
    }
}

결과: scene.usda에서는 HeroSword의 `damageState`를 'damaged'로 선택했기 때문에, 원본 에셋의 여러 버전 중 '부서진' 버전의 칼이 최종적으로 보이게 됩니다.
