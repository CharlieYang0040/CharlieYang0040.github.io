5. Inherits: 속성 상속받기

미리 정의된 '클래스' Prim으로부터 속성을 상속받습니다. 여러 Prim에 공통된 속성을 일괄적으로 적용하고 수정할 때 유용합니다.

① material_class.usda

#usda 1.0
# 이름 앞에 _class_ 를 붙여 클래스임을 명시
class "_class_WoodMaterial"
{
    token materialType = "Wood"
    color3f woodColor = (0.8, 0.5, 0.2)
}

② furniture.usda

#usda 1.0
def "WoodDesk" (
    inherits = @./material_class.usda@
) {}

def "WoodChair" (
    inherits = @./material_class.usda@
) {}

결과: WoodDesk와 WoodChair는 모두 _class_WoodMaterial로부터 재질 속성을 상속받습니다. 나중에 나무 색상을 바꾸고 싶으면 material_class.usda 파일 하나만 수정하면 됩니다.
