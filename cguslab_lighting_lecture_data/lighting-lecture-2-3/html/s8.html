Composition Arcs: 레이어를 조합하는 규칙

여러 레이어에 흩어져 있는 데이터를 조합하여 최종 씬을 만드는 규칙으로, USD의 가장 강력하고 핵심적인 기능입니다.

SubLayers

기본적인 레이어 쌓기 방식. 상위 레이어가 하위 레이어 데이터를 덮어씁니다.

References

외부 파일/프림을 참조합니다. 에셋 조합의 기본 방식입니다.

Payloads

선택적으로 로딩 가능한 Reference. 무거운 데이터 관리에 사용됩니다.

Variants

대체 가능한 데이터 세트. 에셋의 여러 버전을 관리합니다.

Inherits

클래스 프림을 상속받아 여러 프림에 일괄적으로 속성을 적용합니다.

Specializes

Inherits와 유사하지만, 항상 가장 약한 의견을 가집니다.
