3. Payloads: 똑똑하게 로딩하기

Reference와 거의 동일하지만, 씬을 열 때 데이터 로딩 여부를 선택할 수 있는 기능이 추가됩니다. 무거운 에셋을 관리할 때 매우 유용합니다.

① heavy_forest.usda

#usda 1.0
def "Forest"
{
    # 수천 개의 나무가 있다고 가정
    def "Tree_001" { ... }
    def "Tree_002" { ... }
    ...
}

② world.usda

#usda 1.0
def "World"
{
    def "DistantForest" (
        # reference 대신 payload 사용
        payload = @./heavy_forest.usda@
    )
    {
    }
}

결과: world.usda를 열면 DistantForest Prim이 보이지만, 내부 데이터는 로드되지 않은 상태입니다. Maya나 usdview 같은 툴에서 'Load'를 선택해야만 실제 숲 데이터가 메모리에 올라옵니다.
