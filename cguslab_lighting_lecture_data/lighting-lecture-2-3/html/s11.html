2. References: 에셋 불러오기

외부 USD 파일의 특정 Prim을 현재 씬으로 '참조'하여 가져옵니다. 에셋을 재사용하는 가장 일반적인 방법입니다.

① chair_asset.usda

#usda 1.0
def "Chair"
{
    def Cube "Seat" { ... }
    def Cube "Leg1" { ... }
    def Cube "Leg2" { ... }
    def Cube "Leg3" { ... }
    def Cube "Leg4" { ... }
}

② room.usda

#usda 1.0
                    #usda 1.0

                    def "Room"
                    {
                        def "Chair_1" (
                            references = @./chair_asset.usda@
                        )
                        {
                            # 위치 이동
                            # 애셋 자체가 (0,0,0)에 있으므로, 간단하게 transform을 적용할 수 있습니다.
                            double3 xformOp:translate = (10, 0, 0)
                            # 스케일, 회전 등 필요한 다른 transform도 여기에 추가합니다.
                            uniform token[] xformOpOrder = ["xformOp:translate"]
                        }
                    
                        def "Chair_2" (
                            references = @./chair_asset.usda@
                        )
                        {
                            double3 xformOp:translate = (-10, 0, 0)
                            uniform token[] xformOpOrder = ["xformOp:translate"]
                        }
                    }
}

결과: 하나의 의자 에셋(chair_asset.usda)을 두 번 참조하여 방 안에 배치합니다. 각 의자의 위치는 원본 에셋을 건드리지 않고 비파괴적으로 수정할 수 있습니다.
