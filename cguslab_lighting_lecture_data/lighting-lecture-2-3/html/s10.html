1. SubLayers: 레이어 겹쳐 쌓기

가장 기본적인 조합 방식으로, 포토샵 레이어처럼 작동합니다. 리스트의 아래에 있는 레이어가 먼저 깔리고, 위에 있는 레이어가 그 위를 덮어씁니다.

① model.usda (원본)

#usda 1.0
def Sphere "MySphere"
{
    double radius = 2.0
}

② color.usda (덮어쓸 내용)

#usda 1.0
over "MySphere"
{
    color3f[] primvars:displayColor = [(1, 0, 0)]
}

③ main.usda (최종 조합)

#usda 1.0
(
    # 파일 전체에 대한 설정으로 SubLayer를 사용합니다.
    subLayers = [
        @./color.usda@,
        @./model.usda@
    ]
)

결과: model.usda의 구(Sphere) 위에 color.usda의 색상 정보가 덮어씌워져 '빨간색 구'가 나타납니다.
