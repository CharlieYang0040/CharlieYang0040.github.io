6. Specializes: 약하게 상속받기

Inherits와 유사하지만, `specializes`로 상속받은 값은 항상 가장 약한 의견(weakest opinion)으로 처리되어 쉽게 덮어쓸 수 있습니다.

① template_light.usda

#usda 1.0
def "TemplateLight"
{
    float intensity = 1.0
}

② lighting.usda

#usda 1.0
def "KeyLight" (
    specializes = @./template_light.usda@
)
{
    # 이 값은 template의 1.0보다 강함
    float intensity = 10.0 
}

# 만약 KeyLight에 다른 override가 들어오면
# 10.0은 쉽게 덮어써지지만, specializes로
# 가져온 1.0은 항상 그 밑에 깔려있음.

결과: KeyLight는 `TemplateLight`의 속성을 기본값으로 가지지만, 직접 정의한 `intensity = 10.0`이 더 강하므로 최종 밝기는 10이 됩니다. `specializes`는 강력한 오버라이드 체계에서 안전한 기본값을 제공하는 역할을 합니다.
