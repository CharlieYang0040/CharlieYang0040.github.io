5. 그림자와 데이터 패스 활용

후반 작업의 완성도를 높이는 필수 데이터

🌑 그림자 패스 (Shadow Pass)

그림자만 따로 렌더링하여 합성 시 실사 배경 위에 자연스럽게 올리거나, 그림자의 농도, 색상을 자유롭게 조절하는 데 사용됩니다.

  • AOV 설정 창에서 Create > Shadow 로 생성
  • 생성 시 그림자를 만들 조명 지정 (예: Key Light)
  • 결과물: shadow_key_0 와 같은 이름으로 생성
💾 데이터 패스 (Data Pass)

색상 정보가 아닌, 각 픽셀이 가진 위치, 방향 같은 '데이터'를 담고 있는 AOV입니다. 후반 작업에서 3D 효과를 자연스럽게 적용하는 데 핵심적인 역할을 합니다.

  • z (Depth): 안개, 아웃포커싱 (DOF)
  • Nn (Normal): 리라이팅 (Relighting)
  • P (World Position): 3D 오브젝트 추가 합성
