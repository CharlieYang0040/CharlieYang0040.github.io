3. 빛을 분리하다: Light Groups

각 조명의 기여도를 별도 패스로 출력하는 기술

① Light Group 이름 지정

각 조명을 선택하고, Attribute Editor의 Advanced 탭에서 Light Group 파라미터에 고유한 이름을 입력합니다.

Key Light key
Fill Light fill
Rim Light rim
Dome Light env
② Light Group AOV 생성
  1. AOV 설정 창 하단의 LPE Tool 섹션에서 Create > Light Group을 클릭합니다.
  2. Group Name에 ①에서 지정한 이름(예: key)을 입력하고 Create AOV를 누릅니다.

모든 조명 그룹에 대해 반복하면, 아래와 같은 AOV들이 생성됩니다.

lgt_key, lgt_fill, lgt_rim, lgt_env
