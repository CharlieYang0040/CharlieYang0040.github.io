PBR 텍스처 활용: PxrMetallicWorkflow

Substance Painter 텍스처를 손쉽게 연결하기

PxrMetallicWorkflow 노드 사용법

Substance Painter 등에서 사용하는 Metallic 워크플로우 텍스처를 PxrSurface에 손쉽게 연결해주는 전용 노드입니다.

  1. 1단계: PxrMetallicWorkflow 노드를 생성하고 PBR 텍스처를 연결합니다.
    • Base Color 맵 → Base Color
    • Metallic 맵 → Metallic
  2. 2단계: 노드의 결과값을 PxrSurface에 연결합니다.
    • Result Diffuse Color → Diffuse Color
    • Result Specular Face Color → Specular Face Color
    • Result Specular Edge Color → Specular Edge Color
작동 원리

이 노드는 PBR 맵(BaseColor, Metallic)을 입력받아, RenderMan의 Specular 워크플로우에 맞는 값(Diffuse Color, Specular Face/Edge Color)으로 자동 변환하여 출력해줍니다.
