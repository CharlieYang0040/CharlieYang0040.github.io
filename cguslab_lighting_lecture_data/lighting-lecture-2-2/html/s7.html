Object ID vs Material ID 비교

Object ID

  • • 개별 메쉬/객체 분리
  • • 동일 모델 여러 부분 개별 선택
  • • 완벽한 에지 정확도
  • • 애니메이션 자동 추적

Material ID

  • • 동일 머티리얼 객체 그룹화
  • • 질감/표면 기반 선택
  • • 컬러 그레이딩 효율성
  • • 다중 객체 일괄 제어
