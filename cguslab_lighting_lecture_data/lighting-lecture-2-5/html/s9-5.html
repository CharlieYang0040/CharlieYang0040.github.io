📊 딕셔너리 문제: 새로운 메뉴 추가 및 가격 변경 ★★☆☆☆

딕셔너리에 새로운 Key-Value 쌍을 추가하고, 기존 키의 값을 수정하는 방법을 연습합니다.

[요구 사항]
  • '라떼' 메뉴를 4500원의 가격으로 딕셔너리에 새로 추가하세요. (`my_dict['new_key'] = new_value`)
  • '아메리카노'의 가격을 4000원으로 인상(수정)하세요. (`my_dict['existing_key'] = updated_value`)

실행 코드

cafe_menu = {
    "아메리카노": 3500,
    "카푸치노": 4000
}

# 여기에 코드를 작성하세요
# 1. 메뉴 추가
# 2. 가격 변경

print(cafe_menu)

# 실행 결과: {'아메리카노': 4000, '카푸치노': 4000, '라떼': 4500}

모범 답안:

cafe_menu = {
    "아메리카노": 3500,
    "카푸치노": 4000
}

# 1. 메뉴 추가
cafe_menu['라떼'] = 4500

# 2. 가격 변경
cafe_menu['아메리카노'] = 4000

print(cafe_menu)
