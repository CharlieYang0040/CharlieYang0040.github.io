문제: 평균 점수 계산기 ★★★★★

가변 인자(`*args`)를 사용하여 유연한 평균 점수 계산기를 작성하세요.

[요구 사항]
  • 함수 이름은 `calculate_average`로 하세요.
  • 가변 인자(`*args`)를 사용하여 여러 개의 점수를 받을 수 있도록 하세요.
  • 점수들의 평균을 계산하여 반환해야 합니다.
  • 만약 점수가 하나도 입력되지 않았다면 0을 반환하세요.

실행 코드

# 이 아래에 함수를 정의하세요
def calculate_average(*scores):
    # ???

# 실행 예시
print(calculate_average(80, 90, 100))        # 결과: 90.0
print(calculate_average(75, 85, 95, 100))    # 결과: 88.75
print(calculate_average())                   # 결과: 0

모범 답안:

def calculate_average(*scores):
    if not scores:
        return 0
    
    total = sum(scores)
    average = total / len(scores)
    return average

# 실행 예시
print(calculate_average(80, 90, 100))
print(calculate_average(75, 85, 95, 100))
print(calculate_average())
