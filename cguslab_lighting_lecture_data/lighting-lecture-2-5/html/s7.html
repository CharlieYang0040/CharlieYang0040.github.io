while 반복문

특정 조건이 True인 동안 코드를 계속해서 반복합니다. 주로 언제 끝날지 모르는 반복에 사용됩니다.

[기본 예제]

n = 5
while n > 0:
    print(n)
    n = n - 1 # 이 줄이 없으면 무한 반복!
print("발사!")

[실전 문제]

`while`문과 `if`문을 사용하여, 사용자가 "종료"라고 입력할 때까지 사용자가 입력한 말을 그대로 따라 출력하는 '메아리 프로그램'을 만들어보세요.

해답:

while True: # 무한 루프
    user_input = input("입력 ('종료' 시 중단): ")
    if user_input == "종료":
        print("프로그램을 종료합니다.")
        break # 반복문 탈출
    else:
        print(user_input)
