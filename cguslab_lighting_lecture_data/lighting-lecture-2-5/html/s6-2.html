문제: 모든 코디 조합 만들기 🤝

두 리스트의 항목을 조합하여 가능한 모든 코디를 출력하는 코드를 완성하세요.

[문제]

`shirts`와 `pants` 리스트가 주어졌을 때, 중첩 `for` 문을 사용하여 모든 가능한 셔츠와 바지 조합을 만들어 아래 '실행 결과'와 같이 출력하는 코드를 작성하세요.

shirts = ["흰색", "검은색", "파란색"]
pants = ["청바지", "슬랙스", "반바지"]
# 이 아래에 코드를 작성하세요

실행 결과 예시

흰색 셔츠와 청바지 코디
흰색 셔츠와 슬랙스 코디
흰색 셔츠와 반바지 코디
검은색 셔츠와 청바지 코디
... (이하 생략) ...

모범 답안

shirts = ["흰색", "검은색", "파란색"]
pants = ["청바지", "슬랙스", "반바지"]
combinations = []

for shirt in shirts: # 바깥쪽 루프: 셔츠 리스트 순회
    for pant in pants: # 안쪽 루프: 바지 리스트 순회
        combinations.append(f"{shirt} 셔츠와 {pant} 코디")

# 모든 코디 조합 출력
for combo in combinations:
    print(combo)

학습 포인트

  • 첫 번째 리스트의 각 항목에 대해 두 번째 리스트의 모든 항목이 차례로 매칭됩니다.
  • 데이터 간의 모든 경우의 수를 탐색할 때 유용합니다.
