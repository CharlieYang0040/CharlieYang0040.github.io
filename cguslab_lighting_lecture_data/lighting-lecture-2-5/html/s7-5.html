문제: 자판기 프로그램 만들기 🥤 ★★★★★

`while`문을 사용하여 간단한 음료수 자판기 시뮬레이션 프로그램을 작성하세요.

[요구 사항]
  • 초기 자판기 잔액 `money`를 10000원으로 설정하세요.
  • `while money > 0:` 조건으로 자판기에 돈이 남아있는 동안 계속 동작하게 만드세요.
  • 사용자에게 메뉴(1: 콜라-1000원, 2: 사이다-1200원, 3: 종료)를 보여주고 입력을 받으세요.
  • 선택에 따라 잔액을 차감하고, 잔액이 부족하면 "잔액이 부족합니다."를 출력하세요.
  • 사용자가 3번을 선택하면 `break`로 프로그램을 종료하세요.

모범 답안:

money = 10000

while money > 0:
    print(f"[현재 잔액: {money}원] 메뉴를 선택하세요 (1:콜라-1000원, 2:사이다-1200원, 3:종료)")
    choice = input()

    if choice == '1':
        if money >= 1000:
            money -= 1000
            print("콜라를 구매했습니다.")
        else:
            print("잔액이 부족합니다.")
    elif choice == '2':
        if money >= 1200:
            money -= 1200
            print("사이다를 구매했습니다.")
        else:
            print("잔액이 부족합니다.")
    elif choice == '3':
        print("자판기 작동을 중지합니다.")
        break
    else:
        print("잘못된 메뉴입니다.")
