문제: 숫자 맞추기 게임 (Up & Down) 🎲 ★★★☆☆

컴퓨터가 정한 숫자를 사용자가 맞출 때까지 힌트를 주며 게임을 진행하는 프로그램을 작성하세요.

[요구 사항]
  • `random` 모듈을 사용하여 1부터 100 사이의 정답 숫자를 만드세요.
  • 사용자가 정답을 맞출 때까지 `while`문으로 게임을 계속 진행하세요.
  • 사용자가 입력한 숫자가 정답보다 크면 "다운!", 작으면 "업!"이라고 힌트를 주세요.
  • 정답을 맞추면 "정답입니다!"를 출력하고 `break`로 루프를 종료하세요.

실행 코드

import random

answer = random.randint(1, 100)
print("1부터 100 사이의 숫자를 맞춰보세요!")

모범 답안:

import random

answer = random.randint(1, 100)
print("1부터 100 사이의 숫자를 맞춰보세요!")

while True:
    try:
        guess = int(input("숫자를 입력하세요: "))
        if guess > answer:
            print("다운!")
        elif guess < answer:
            print("업!")
        else:
            print("정답입니다!")
            break
    except ValueError:
        print("숫자만 입력해주세요.")
