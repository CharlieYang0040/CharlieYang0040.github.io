📊 딕셔너리 문제: 모든 키와 값 출력하기 ★★★☆☆

`.items()` 메소드를 사용하여 딕셔너리의 모든 내용을 형식에 맞게 출력하세요.

[요구 사항]
  • `for` 반복문과 `.items()` 메소드를 사용하여 딕셔너리의 모든 키와 값을 한 번에 가져와야 합니다.
  • "상품: [이름], 재고: [수량]개" 형식으로 출력해야 합니다.

실행 코드

inventory = {
    "사과": 20,
    "바나나": 15,
    "오렌지": 30
}

# 여기에 코드를 작성하세요
# ???

# 실행 결과:
# 상품: 사과, 재고: 20개
# 상품: 바나나, 재고: 15개
# 상품: 오렌지, 재고: 30개

모범 답안:

inventory = {
    "사과": 20,
    "바나나": 15,
    "오렌지": 30
}

for product, stock in inventory.items():
    print(f"상품: {product}, 재고: {stock}개")
