문제: 2차원 리스트 순회하기 🏁

주어진 2차원 리스트의 모든 요소를 순회하며 출력하는 코드를 완성하세요.

[문제]

`matrix`라는 2차원 리스트가 주어졌을 때, 중첩 `for` 문을 사용하여 모든 요소를 순회하고 아래 '실행 결과'와 같이 출력하는 코드를 작성하세요.

matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
# 이 아래에 코드를 작성하세요

실행 결과 예시

현재 행: [1, 2, 3]
  요소: 1
  요소: 2
  요소: 3
현재 행: [4, 5, 6]
... (이하 생략) ...

모범 답안

matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

# 2차원 리스트의 모든 요소를 하나씩 출력
for row in matrix: # 바깥쪽 루프: 각 행(row)에 접근
    print(f"현재 행: {row}")
    for element in row: # 안쪽 루프: 각 요소(element)에 접근
        print(f"  요소: {element}")

학습 포인트

  • 바깥 루프에서 큰 리스트의 요소(작은 리스트)를, 안쪽 루프에서 작은 리스트의 요소들을 꺼내는 구조입니다.
  • 데이터 분석이나 게임 개발 등에서 행렬 데이터를 처리할 때 필수적인 패턴입니다.
