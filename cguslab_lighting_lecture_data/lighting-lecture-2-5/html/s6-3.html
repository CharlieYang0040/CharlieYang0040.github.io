문제: 별 찍기 패턴 ⭐

중첩 `for` 문을 사용하여 주어진 모양대로 별을 출력하는 코드를 완성하세요.

[문제]

중첩 `for` 문을 사용하여 아래 '실행 결과'와 같이 5층짜리 직각 삼각형 모양의 별을 찍는 코드를 작성해보세요.

실행 결과 예시

*
**
***
****
*****

모범 답안

# 5층짜리 별표 직각 삼각형 만들기
for i in range(1, 6): # 바깥쪽 루프: 층(행)을 제어 (1층부터 5층까지)
    for j in range(i): # 안쪽 루프: 각 층에 찍을 별의 개수를 제어
        print("*", end="") # 별을 찍고 줄바꿈을 하지 않음
    print() # 한 층의 별을 다 찍고 나면 줄바꿈

학습 포인트

  • 안쪽 루프의 반복 횟수(range(i))가 바깥쪽 루프의 현재 값(i)에 따라 동적으로 변합니다.
  • print() 함수의 end 매개변수를 이용해 출력 형태를 제어할 수 있습니다.
